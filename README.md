# Image Classification Paper-Rock-Scissors

Proyek ini membangun model klasifikasi gambar untuk mengenali tiga kelas: `paper`, `rock`, dan `scissors` menggunakan TensorFlow/Keras.

## Struktur Proyek
- `Image_classification_paper-rock-scissors.ipynb`: Notebook utama untuk preprocessing, training, evaluasi, dan inferensi.

## Dataset
Dataset berisi gambar tangan dengan label:
- `paper`
- `rock`
- `scissors`

## Cara Menjalankan
1. Buat environment Python.
2. Install dependency yang dibutuhkan notebook.
3. Jalankan notebook `Image_classification_paper-rock-scissors.ipynb` dari awal hingga akhir.

Contoh perintah:

```bash
pip install tensorflow matplotlib numpy scikit-learn jupyter
jupyter notebook
```

## Hasil
Notebook menghasilkan model klasifikasi yang dapat memprediksi kelas gambar tangan `paper`, `rock`, atau `scissors`.
